## Setup

$ make install

Options:
*    support for three input file types: `json` `yml` `ini`  and three output types: `plain` `diff` `json`
*    `-V, --version` output the version number
*    `-h, --help` help page   
*    `-f, --format [type]` output format
Compares two files .json and shows a difference
